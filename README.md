__"Home library" - программное средство для хранения домашней литературы в структурированном виде.__
-
Имеет следующий функционал: 
- Регистрация;
- Авторизация;
- Выбор темы приложения (тёмная/светлая);
- Добавление новой книги в библиотеку;
- Редактирование книги;
- Удаление книги;
- Поиск по списку книг;
- Сортировка по названию книги, автору, даты издания, жанру и номеру полки;
- Отображение статистики жанров литературы.
